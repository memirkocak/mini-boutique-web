# GreenMarket - Mini Boutique Web

## Description

GreenMarket est une boutique en ligne éco-responsable proposant des cosmétiques naturels, vêtements recyclés et accessoires zéro déchet. Le site présente un catalogue de produits avec système de tri et pages de détail.

Maquette Figma : https://www.figma.com/design/WYUSv7WxEyYVcjhtmNZEfT/mini-boutique-web?node-id=0-1&p=ft=KVDSL6E1dWf5D7Rn-0

## Installation
Lien github : https://github.com/memirkocak/mini-boutique-web

1. Cloner ou télécharger le projet
2. Ouvrir le dossier dans un terminal puis "lancement"
