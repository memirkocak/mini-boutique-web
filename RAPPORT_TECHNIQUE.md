# Rapport Technique - GreenMarket

## 1. SEO 

- Balises HTML : Utilisation de "<header>", "<nav>", "<main>", "<section>", "<article>", "<footer>"
- Hiérarchie des titres: Respect de la structure H1 H2 H3 sur toutes les pages
- ARIA : aria-labelledby pour lier les sections à leurs titres
- Page d'accueil : Meta description présente
- Page produits : Meta description présente
- Page détail produit : Meta description présente
- Code structuré : Séparation HTML/CSS/JS, approche mobile-first

## 2. Accessibilité

- Focus visible : Styles :focus et :focus-visible avec outline
- Tous les éléments interactifs : Accessibles au clavier (liens, boutons, select)
- Texte principal : (#2c3e2d sur #ffffff) 
- Boutons : (blanc sur vert foncé) 
- Footer : (blanc sur vert foncé) 
- Attributs alt : alt pertinents sur toutes les images
- Images dynamiques : Alt mis à jour via JavaScript avec le titre du produit
- Balises HTML : Bonne utilisation des balises sémantiques
- Responsive : mobile-first / desktop

## 3. Choix techniques

- HTML : Structure sémantique, balises
- CSS : mobile-first / desktop
- JavaScript :code modulaire et maintenable
- W3C : Code validé avec le validateur W3C

## 4. Points forts

Accessibilité complète : Navigation clavier, contrastes , structure sémantique
SEO optimisé : Meta descriptions, hiérarchie des titres
Code propre : mobile-first, maintenable
Performance :code optimisé, pas de dépendances lourdes
Conformité : W3C validé
